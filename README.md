# Social Media Use and Affective Polarization

### A longitudinal quantitative analysis using R

This repository presents the code, methodology and main results of my Bachelor's Thesis in Sociology at the University of Salamanca.

The study examines the relationship between informational social media use and affective polarization among young adults in Spain, using longitudinal survey data from the TriPol España project.
